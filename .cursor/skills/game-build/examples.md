# Example prompts

## Card party + print-and-play + classroom

```
@game-build

Game type: card-party
Deliverable: print-and-play
Audience: classroom
Art: flat-vector

Brief: 4-8 players, 20 minutes, ocean conservation theme. Generate friendly marine animal art on each prompt card. Cooperative scoring optional.
```

## Board abstract + rules only

```
@game-build

Game type: board-abstract
Deliverable: rules-only
Audience: hobby

2 players, 30 minutes, perfect information, control most spaces on a 5x5 grid.
```
